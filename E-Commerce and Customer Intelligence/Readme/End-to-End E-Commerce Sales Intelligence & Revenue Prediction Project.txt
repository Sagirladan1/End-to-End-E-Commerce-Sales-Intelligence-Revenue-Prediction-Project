# E-Commerce Sales Analysis & Business Intelligence Project

## Project Overview

This project focuses on analyzing an e-commerce sales dataset using:

- Excel for data cleaning
- MySQL for data analysis
- Power BI for dashboard visualization
- Python (Jupyter Notebook) for exploratory data analysis and visualization

The objective of this project is to transform raw business data into meaningful insights that support business decision-making.

---

# Business Problem

The company needed to understand:

- Revenue performance
- Product sales trends
- Customer spending behavior
- Sales representative performance
- Regional sales performance
- Operational delivery performance

The raw dataset contained:
- Missing values
- Duplicate records
- Inconsistent date formats
- Invalid entries
- Incorrect revenue calculations

The project solved these issues through a complete data analytics workflow.

# Sample SQL Query

```sql
-- Analyze total revenue generated by each country
-- Helps identify top-performing regions based on sales revenue

SELECT 
    Country,
    SUM(Clean_Revenue) AS Revenue
FROM ecommerce_cleaned
GROUP BY Country
ORDER BY Revenue DESC;
```
